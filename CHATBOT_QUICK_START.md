# 🚀 Quick Start - Chatbot AI HijabKu

## Fitur Chatbot Sudah Ditambahkan! ✅

Chatbot AI telah berhasil diintegrasikan ke seluruh website HijabKu.

---

## 🎯 Cara Menggunakan (Super Mudah!)

### 1️⃣ Buka Website
Buka salah satu halaman website HijabKu:
- `index.html` (Login)
- `dashboard.html` (Dashboard)
- `katalog.html` (Katalog Produk)
- `pesanan.html` (Pesanan)
- `riwayat.html` (Riwayat)
- `help.html` (Bantuan)

### 2️⃣ Cari Tombol Chatbot
Lihat di **pojok kanan bawah** layar → Ada tombol **ungu dengan icon chat** 💬

### 3️⃣ Klik Tombol Chat
- Klik tombol tersebut
- Window chatbot akan muncul
- Bot akan menyapa Anda!

### 4️⃣ Mulai Bertanya
Ketik pertanyaan atau klik quick action buttons:
- 📦 Lihat Produk
- 🛒 Status Pesanan  
- ❓ Bantuan

### 5️⃣ Kirim Pesan
- Tekan **Enter** atau
- Klik tombol **Send** (icon pesawat)

## 🐛 Troubleshooting

### Chatbot Tidak Muncul?
1. ✅ Pastikan file `chatbot.js` dan `chatbot.css` ada
2. ✅ Refresh browser (F5)
3. ✅ Check console browser (F12) untuk error

### Tombol Tidak Berfungsi?
1. ✅ Pastikan JavaScript aktif di browser
2. ✅ Clear cache browser
3. ✅ Coba browser lain
