# 📑 Index - HijabKu Project

## 🎯 Quick Start

**Untuk langsung mencoba aplikasi:**
1. Buka file `index.html` di browser
2. Login dengan: `admin@hijabku.com` / `admin123`

1. Buka file `presentasi.html` di browser
2. Gunakan arrow keys atau tombol navigasi
